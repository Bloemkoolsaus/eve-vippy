If you do not yet have an account on VIPPY, register here to create one.

After registration, add your characters to your profile by using the EVE API keys to gain access to your corporation or alliance intell.

Please note that this API key must be a new one. Because of how the API system works, previously used keys often fail.


Please note that you only need one VIPPY account.
You can use multiple characters on one VIPPY account!

You need atleast one valid API key for a valid character to grant you access to VIPPY. Once you have an authorized VIPPY account, you can use any pilot you want, including out-of-corp alts.

To use an alt, simply open VIPPY via the ingame browser of the alt pilot, and login in to VIPPY with your VIPPY login. You can have multiple VIPPY sessions open for multiple pilots using the same VIPPY account.
