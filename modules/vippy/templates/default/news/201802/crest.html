Until today, Vippy was (still) using the CREST and XML api's. These have been deprecated for a while now and CCP is shutting them down in May 2018. The replacement is ESI and is supposed to be faster, more reliable and have more features (including write access).

Vippy has now switched to the new ESI api, and because of that all CREST tokens have become invalid and you have to generate a new token for every character you have on Vippy. You can do this in your profile.
