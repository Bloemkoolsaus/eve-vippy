Hello explorers,

Up untill today, Vippy was (still) using the CREST and XML api's. These technologies have been deprecated by CCP for a while now, and CCP is shutting them down in May 2018. The are replaced by ESI. This new api is supposed to be faster, more reliable and it has more features (including write access to some degree).

Vippy has now switched to the new ESI api and because of that all your CREST tokens have become invalid. This means you have to generate a new ESI token for every character you have/want on Vippy. You can do this in your profile.
