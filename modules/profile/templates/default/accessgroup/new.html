Setting up a new access group

Vippy access is based on access groups. You can set wich corporations or alliances can see your access group.

Using Vippy and RENT

You can use VIPPY for free for the first month! This is a trial period to see if you like Vippy.
VIPPY does not run itself, the rent is mostly to help pay for server costs. After the trial ends, you will be expected to pay rent.
Please contact Bloemkoolsaus ingame if you want to go for a full subscription.
Prices range from 100m for a small corp (10+ active users, not active toons) up to a maximum of 500m for a large corp (50+)
