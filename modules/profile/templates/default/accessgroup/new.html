Using Vippy and subscription

You can use VIPPY for free for the first month! This is a trial period to see if you like Vippy.
VIPPY does not run itself, the rent is mostly to help pay for server costs. After the trial ends, you will be expected to pay a subscription fee.
The subscription system is about to be refactored. Please read this page for more information.
