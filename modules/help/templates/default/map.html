Adding a wormhole signature

If you add a WH signature, VIPPY can create a temporary unmapped wormhole. You have to decide on a name for the new wormhole, and put that name in the `info` column of the signature.
If the name of the temporary wormhole is correct, as soon as a system is mapped that gets the name of a temp one, the temporary system will be replaced by the new mapped one.
The naming works as follows:
    FROM-TO other info
The name of the new system will be grabbed out of the FROM-TO part. Always use FROM dash TO.
Any words after the FROM-TO will not be included in the new name of the temporary wormhole.
However, certain keywords are defined:
  frig   = Will make it a frigate only wormhole
  eol   = Will mark the wormhole end-of-life
  reduced   = Will mark the wormhole mass reduced
  crit   = Will mark the wormhole mass crit

System Bar

Gray system bars indicate whether or not a system requires scanning or scouting.
If you are looking for something to do, start in systems with a bar!
  
FULL BAR
This system has not been scanned.
  
HALF BAR
This system was scanned, but it was more then 2 hours ago. New things may have popped up!
